# 需求文档 — 超天酱日常推文小站

> 版本：v0.8 · 最后更新：2026-06-13

---

## 功能需求

### F1: 双人格推文生成 ✅
- **描述**：DeepSeek API 同时产出一条「超天酱推文」和一条「糖糖日记」
- **超天酱风格**：元气可爱、大量 emoji、对粉丝说话、偶尔无意露脆弱
- **糖糖风格**：阴暗疲惫、零 emoji、短句碎片、自言自语
- **输入**：可选主题/心情关键词，留空则随机选题（90% 通用 + 10% 专属）
- **输出**：两条纯文本推文，各 1-3 行，~280 字符以内
- **实现**：`generator.py` — 独立可运行，`POST /api/generate` — 服务器端点

### F2: 推文 Feed 展示 ✅
- **描述**：以时间线形式展示历史推文，Poketter 风格
- 超天酱推文：粉色卡片 + 认证徽章 + 固定像素头像 + 随机自拍配图
- 糖糖日记：暗色卡片 + 去饱和头像 + glitch hover 效果 + 随机自拍配图
- 每条标注日期、时间、主题标签
- 最新推文在最上方
- **实现**：`static/index.html`

### F3: 推文持久化 ✅
- **描述**：生成的推文自动保存到 `data/feed.json`
- 启动时自动加载历史 feed
- 最多保留 50 条，超出截断头部
- **实现**：`feed.py`

### F4: 视觉风格 ✅
- **描述**：粉紫像素风 UI，还原游戏 Poketter 质感
- 配色：深紫背景 + 粉色主色调 + 紫色辅助
- 像素风字体（DotGothic16）
- CRT 扫描线效果 + Win95 视窗风格
- 游戏原版头像（`icon_cho/ame`，51×51 像素，卡片 header 固定）
- 游戏原版自拍配图（`cho_*.png` / `ame_*.png`，17 张，附在推文正文里）
- 游戏 HUD 状态栏（Followers / Affection / Stress / Darkness）
- 桌面图标行（KAngel / Ame / JINE）
- **实现**：`static/index.html` + `static/img/`

### F5: JINE 私信 Tab ✅
- **描述**：糖糖给阿P的私密聊天界面
- 即时通讯气泡风格（还原游戏内 JINE App）
- 内容由 v2 时间线生成器自动产出
- 三种视图 Tab 切换：All / Diary / JINE
- **实现**：`static/index.html` v2.0

### F6: 直播画面小组件 ✅
- **描述**：假装超天酱正在直播的小组件
- CSS 呼吸灯 LIVE 角标（纯 CSS 脉冲动画）
- 虚拟观看人数随机抖动（JS setInterval）
- 滚动弹幕（6 条预设弹幕随机刷新滚动）
- **实现**：`static/index.html` v2.0

### F7: 点击发表新动态 ✅
- **描述**：「戳一戳超天酱 ✨」按钮
- 单击 → `POST /api/release` → 从 hidden_pool 释放一条预存推文
- hidden_pool 耗尽时自动触发生成
- 无文本输入框（维持艺术装置感）
- **实现**：`server.py` + `static/index.html` v2.0

### F6.5: 弹幕垃圾桶 ✅
- **描述**：还原游戏「消除负面评论」玩法
- 弹幕分为普通（白色）和负面（红色），负面弹幕可点击
- 点击负面弹幕 → 飞入垃圾桶消除动画 → Stress −3
- 右下角垃圾桶计数器 + 震动反馈
- 12s 自动清理残留负面弹幕
- **实现**：`static/index.html` v2.0

### F8: JINE 互动聊天 🔜
- **描述**：玩家在 JINE Tab 中发送游戏表情包，糖糖 AI 实时回复
- 表情包池：游戏内 sticker（从 Steam 提取或预设像素表情）
- 糖糖回复风格：碎句、省略号、试探、依赖——与现有 JINE 一致
- 交互流程：点表情包 → 发送 → 调 API → 糖糖回复气泡出现
- 维持窥探感：不是「聊天工具」，是「你以阿P的身份在回她」

## 非功能约束

### N1: 技术约束
- 必须使用 conda 环境 `deepseek_v4_api`
- 必须使用 DeepSeek API（deepseek-v4-pro）
- UnityPy 可用于素材提取（非运行时依赖，提取后即可卸载）
- 不依赖 GPU
- 启动速度 < 5 秒

### N2: 内容约束
- 推文内容必须符合角色人设
- 不得生成政治敏感、色情暴力内容
- 糖糖阴暗面控制在「无力感 + 自我怀疑 + 渴望陪伴」尺度
- JINE 私信内容同样遵守上述约束

### N3: 性能约束
- 单次生成响应时间 < 10 秒
- API 调用失败时自动重试（最多 3 次，指数退避）
- 前端 30s 自动轮询新推文

### N4: 目录约束
- 项目位于 `e:\Desktop\Deepseek_V4_API\amechan-daily\`
- 游戏素材从 `E:\Steam\steamapps\common\NEEDY GIRL OVERDOSE\` 提取
- 遵循三文档同步规则
