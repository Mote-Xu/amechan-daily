# JINE 原版 Stamp PNG

将游戏原版 8 张猫咪 stamp 放到此目录，CSS pixel cat 会自动作为 fallback。

## 来源：Spriters Resource

访问以下 URL 下载 JINE 精灵图集（270×910 竖条，含全部 8 张 stamp）：

**https://www.spriters-resource.com/pc_computer/needystreameroverload/asset/200974/**

（由 WordedPuppet 提交，2023-07-10）

## 需要的文件名

下载后将图集按 stamp 切割，保存为以下文件名：

| 文件名 | 对应 stamp | 含义 |
|--------|-----------|------|
| `stamp_good.png` | 好 | 称赞/好评 |
| `stamp_strong.png` | 太强了 | 太厉害了 |
| `stamp_cry.png` | 嘤嘤 | 委屈/撒娇 |
| `stamp_ignore.png` | 关我屁事 | 不关我事 |
| `stamp_sorry.png` | 抱歉抱歉 | 道歉 |
| `stamp_dead.png` | 原地去世 | 死了/躺平 |
| `stamp_love.png` | 永远爱你 | 爱心 |
| `stamp_yeah.png` | 啊对对 | 敷衍/附和 |

## 备用方案：AssetStudio

如果 Spriters Resource 无法访问，使用 AssetStudio 从游戏文件解包：
1. 下载 AssetStudio：https://github.com/Perfare/AssetStudio
2. 加载 `Windose_Data/` 文件夹
3. 搜索 `JineStampView2D` 或 `Stamp2D` 相关 Sprite
4. 导出 stamp PNG
