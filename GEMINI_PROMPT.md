## 一、项目概览

模拟《主播女孩重度依赖》糖糖/超天酱的社交媒体。纯前端 HTML + Python http.server 后端，DeepSeek V4-pro。

三层内容：Poketter 推博（超天酱推文 + 糖糖日记）、JINE 私聊（糖糖↔阿P）、直播间弹幕。

## 二、架构（v4 无状态）

已彻底无状态化，为公网部署做准备：
- 前端 localStorage 管理所有数据（时间线、pool、JINE 历史）
- 后端纯计算：接收参数 → 拼 prompt → 调 DeepSeek → 返回 JSON
- 不读不写任何文件
- ThreadingHTTPServer 多线程
- 限频代码已有（本地关闭）
- 部署目标：Vercel(前端) + 轻量云服务器(后端) + Cloudflare(HTTPS)

## 三、Prompt 架构（v3.3 Menhera）

所有 JINE prompt 使用 XML 标签结构，核心设定：
- 地雷系 (Menhera) 人设，不是廉价傲娇
- 糖糖阿P同居
- 硬禁：哼/揍你哦/笨蛋、点评贴图、点评说话方式、找借口
- 关键规则：猫咪贴图 = 阿P本人在表达情绪，直接回应情绪不分析表情包
- 文字 temp 0.85，贴图 temp 0.85，F7 释放 temp 0.5
